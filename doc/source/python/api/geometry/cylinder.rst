Cylinder
========

.. class:: Geometry.Cylinder(radius=1, height=2, sides=24)

   *Inherits from*: :class:`Geometry.ColorfulObject`

   .. method:: setRadius(radius)
      
      Sets the radius of the cylinder. Must be positive.


   .. method:: setHeight(height)
      
      Sets the height of the cylinder. Must be positive.


   .. method:: setNumberOfSides(sides)

      Sets the number of sides used to render a circular cross-section
      of each cylinder. Must be a positive integer.


   .. method:: radius()

      Returns the radius of the cylinder.


   .. method:: height()

      Returns the height of the cylinder.


   .. method:: numberOfSides()

      Returns the number of sides used to render a circular cross-section
      of each cylinder.
